# React Todo List

## 開発準備

### Node.jsのダウンロード
公式サイト <a href="http://nodejs.org/">http://nodejs.org/</a> から

### パッケージのインストール
package.jsonのあるフォルダで、下記コマンドを実行。
```
$ npm install
```

### gulpの実行
gulpfile.jsのあるフォルダで、下記コマンドを実行。
```
$ gulp
```

### ローカルでページ確認
gulpを実行するとサーバーが立ち上がるので、「<a href="http://localhost">http://localhost</a>」にアクセスして確認。
